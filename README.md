# progcart: Programação aplicada à Engenharia Cartográfica

Professor: Maurício C. M. de Paulo - D.Sc.

Curso distribuído sob licença MIT para aprimorar a qualidade do conteúdo. Caso tenha sugestões de melhoria, faça um pull request. Não sabe como? [Aprenda aqui](aulas/GIT.pdf).

Os slides estão disponíveis na [pasta aulas](aulas/).

O código tex para gerar os pdfs dos slides estão na pasta [aulas/modelo_latex](aulas/modelo_latex/)

A compilação dos arquivos tex e a movimentação para a pasta "aulas" é feita pelo script aulas/modelo_latex/move_pdfs.py. Os pdfs são movidos criados na pasta "aulas/modelo_latex/build" e movidos para a pasta "aulas".

```bash
cd aulas/modelo_latex/
python move_pdfs.py
```
